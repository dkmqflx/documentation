# 문서

![feature-sliced-banner](/img/banner.jpg)

**Feature-Sliced Design**(FSD)는 프론트엔드 애플리케이션을 구조화하기 위한 아키텍처 방법론입니다. 간단히 말해, 코드를 조직하는 규칙과 컨벤션 모음입니다. 이 방법론의 주요 목적은 끊임없이 변화하는 비즈니스 요구사항 속에서도 프로젝트를 더 이해하기 쉽고 체계적으로 만드는 것입니다.
